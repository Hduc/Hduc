## Lệnh trong linux
```
apt-get update

ln -s /usr/share/zoneinfo/Asia/Ho_Chi_Minh /etc/localtime

apt-get install <name>

```
### tạo folder
`mkdir <path>`

### xóa folder 
`rm -rf <folder>`

### di chuyển folder || hoặc đổi tên
`mv <path from> <path to>`

### Kiểm tra CPU
`cat /proc/cpuinfo`

### Kiểm tra RAM trống
`free -h`
Chi tiết hơn:`cat /proc/meminfo`

### Kiểm tra ổ cứng
`df -h`
